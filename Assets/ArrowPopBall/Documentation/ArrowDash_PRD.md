# Arrow Dash (애로우 대시) PRD

## 1. 개요

### 1.1 기능 요약
**Arrow Dash**는 특정 방향(동/서/남/북)을 바라보는 모든 화살표를 한 번에 탈출시키는 아이템 기능입니다.

### 1.2 핵심 가치
- **전략적 선택**: 막힌 퍼즐 상황에서 돌파구 제공
- **시간 단축**: 여러 화살표를 한 번에 처리하여 플레이 효율성 증가
- **수익화 요소**: 아이템 구매를 통한 수익 모델

---

## 2. 기능 상세

### 2.1 Arrow Dash 아이템 시스템

| 항목 | 내용 |
|------|------|
| 아이템 타입 | 소모성 아이템 |
| 초기 지급량 | 계정 생성 시 3개 |
| 저장 방식 | PlayerPrefs (추후 서버 연동 예정) |
| 획득 방법 | 인앱 구매 (추후 구현) |

### 2.2 사용 조건
- 보유한 Arrow Dash 아이템이 1개 이상일 때 사용 가능
- 레벨 내 사용 횟수 = 보유 아이템 개수

### 2.3 탈출 규칙
| 규칙 | 설명 |
|------|------|
| 대상 선정 | 선택한 방향과 동일한 HeadDirection을 가진 모든 화살표 |
| 충돌 처리 | **충돌 무시** - 다른 화살표에 막혀있어도 탈출 |
| 동시 탈출 | 모든 대상 화살표가 동시에 탈출 시작 |
| 풍선 팝 | 일반 탈출과 동일하게 HomingArrow → 풍선 팝 |

## 3. UI/UX 플로우

### 3.1 진입점
- **위치**: BottomBarUI (화면 하단)
- **버튼**: Arrow Dash 아이콘 + 보유 개수 표시

### 3.2 Arrow Dash UI 활성화 시

```
┌─────────────────────────────────────┐
│  [Arrow Dash 설명 영역]        [X]  │  ← 상단: 설명 + 닫기 버튼
│                                     │
│  "선택한 방향의 모든 화살표가       │
│   한 번에 탈출합니다!"              │
├─────────────────────────────────────┤
│                                     │
│         (퍼즐 영역 - 줌 아웃)       │
│                                     │
├─────────────────────────────────────┤
│                                     │
│      [←]  [↑]  [↓]  [→]            │  ← 하단: 방향 버튼 (서/북/남/동)
│      West North South East          │
│                                     │
│         보유: 3개                   │
└─────────────────────────────────────┘
```

### 3.3 카메라 동작
| 단계 | 동작 |
|------|------|
| UI 진입 | 카메라 줌 아웃 + 퍼즐판 중앙 정렬 |
| UI 종료 | 원래 카메라 위치 및 줌 레벨 복원 |

### 3.4 인터랙션 차단
- Arrow Dash UI 활성화 중에는 인게임 영역(화살표 탭 등) 상호작용 불가

## 4. 방향 버튼 동작

### 4.1 버튼별 기능

| 버튼 | 방향 | 대상 HeadDirection |
|------|------|-------------------|
| ← | West (서) | ArrowDirection.Left |
| ↑ | North (북) | ArrowDirection.Up |
| ↓ | South (남) | ArrowDirection.Down |
| → | East (동) | ArrowDirection.Right |

### 4.2 버튼 클릭 시 처리

```
버튼 클릭
    ↓
해당 방향 화살표 존재 확인
    ↓
[있음]                      [없음]
    ↓                           ↓
아이템 1개 소모              안내 메시지 표시
    ↓                       "해당 방향에 화살표가 없습니다"
모든 대상 화살표 동시 탈출
    ↓
HomingArrow 생성 → 풍선 팝
    ↓
Arrow Dash UI 자동 종료
    ↓
카메라 원상 복귀
```

## 5. 기술 명세

### 5.1 데이터 저장
```csharp
// PlayerPrefs 키
"ArrowDashCount" // int, 보유 개수

// 초기값 설정 (최초 실행 시)
if (!PlayerPrefs.HasKey("ArrowDashCount"))
{
    PlayerPrefs.SetInt("ArrowDashCount", 3);
}
```

### 5.2 화살표 필터링
```csharp
// 특정 방향 화살표 조회
List<ArrowController> GetArrowsByDirection(ArrowDirection direction)
{
    return _arrows.Where(a =>
        a.HeadDirection == direction &&
        a.CanLaunch
    ).ToList();
}
```

### 5.3 동시 탈출 처리
```csharp
// 충돌 무시 플래그를 설정하여 탈출
foreach (var arrow in targetArrows)
{
    arrow.LaunchWithoutCollision(); // 새로운 메서드 필요
}
```

## 6. 예외 처리

| 상황 | 처리 |
|------|------|

| 이미 이동 중인 화살표 | 대상에서 제외 (CanLaunch = false) |
